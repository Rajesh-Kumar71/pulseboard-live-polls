# PulseBoard - Live Polls For Feedback

PulseBoard is a MERN stack hackathon project for Web Dev Cohort 2026.

It allows users to create polls, share public links, collect responses and view live analytics.

## Tech Stack

- React + Vite
- Node.js
- Express.js
- MongoDB
- Socket.IO
